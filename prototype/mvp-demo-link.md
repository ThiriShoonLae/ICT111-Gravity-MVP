## Testing Notes
Testers should follow tasks T1–T6 from `/docs/experiment-script.md`:
1. Read the homepage header and say what Gravity is for.
2. Post a listing (desk, 800 THB, Good, Dorm A) — expect a success message and the desk appearing in the list.
3. Find a textbook under 300 THB using search or the category filter.
4. Click it to open the detail — note the RSU verified badge.
5. Explain the Available / Reserved / Sold statuses.
6. Press "Request contact" on any available item and explain what happens next.

Records save in the tester's browser only. Facilitator: press **Reset demo data** between testers so everyone starts from the same seed.
